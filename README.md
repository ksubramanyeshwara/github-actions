# Github Actions

This repository contains notes, examples, and CICD pipelines documenting my journey of learning Github Actions.

## 🎯 Learning Objectives

- Understanding Workflow Fundamentals
- Configuring Event Triggers
- Managing Jobs and Steps Execution
- Working with Runners (GitHub-hosted & Self-hosted)
- Using Reusable Actions from Marketplace
- Securing workflows with Secrets & Environment Variables
- Implementing Continuous Integration (CI) pipelines
- Building Continuous Deployment (CD) workflows
- Integrating Docker in CI/CD pipelines
- Managing Workflow Dependencies and Execution Order
- Handling Artifacts and Data Sharing between Jobs
- Optimizing workflows using Caching Strategies
- Applying Matrix Strategy for Multi-Environment Testing
- Enforcing Security Best Practices and Permissions

## 🧠 Prerequisites

- Linux basics and command-line usage
- Basic knowledge of Git and GitHub
- YAML Basics
- Docker Basics
